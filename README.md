# Mama AI — Personal Assistant for Working Moms

A beautiful, warm React app powered by the Claude API. Includes an AI chat assistant with 5 specialized modes, a task manager with categories and priorities, and a daily dashboard.

## Setup

### 1. Install dependencies
```bash
npm install
```

### 2. Add your Anthropic API key
Open `src/anthropic.js` and replace:
```js
const API_KEY = 'YOUR_ANTHROPIC_API_KEY'
```
Get your key at https://console.anthropic.com

> **Security note:** For production, move the API call to a backend server (Node/Express, Python/FastAPI, etc.) so your API key is never exposed in the browser.

### 3. Run the app
```bash
npm run dev
```

Visit http://localhost:5173

## Build for production
```bash
npm run build
```

## Features

### Today Dashboard
- Daily overview with task progress bar
- Quick-prompt shortcuts to common needs
- Mood check-in that routes to chat

### AI Chat — 5 Modes
- **General** — All-purpose life assistant
- **Work** — Professional coach (emails, decisions, boundaries)
- **Family** — Parenting, scheduling, relationships
- **Wellness** — Realistic self-care for time-pressed moms
- **Finance** — Budgeting, saving, planning

### Task Manager
- Add tasks with category + priority
- Filter by category or done/pending
- Priority color coding (high/medium/low)
- Categories: Family, Work, Self-care, Home, Finance
